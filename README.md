# Flee Chrome Plugin
一个简单易用的`Chrome`插件，包含大部分常见的编码和其他功能。
## ✨功能
#### 加密
1. `MD5`加密
#### 编码、

1. `Base 16`
2. `Base 32`
3. `Base 58`
4. `Base 62`
5. `Base 64`
6. `URL` 编码
7. `ASCII` 编码
8. 摩斯（`Morse`）编码
9. `Unicode` 编码
10. `UTF-8` 编码

#### 其他功能
1. `base64`转文件
    -  如图像`base64`形式转图像文件
2. `hex`转文件
    - 如`504b0304....`转为`zip`压缩包文件
3. 进制转换
